# Implementar feature — Ceopet — TransformaTech (TTech)

1. `Documentação/requisitos-tecnicos/RT-XXX-*.md` com **CAT-*** definidos.
2. `Documentação/requisitos-negocio/RN-XXX-*.md` para contexto de negócio.
3. Plano no RT ou autorização explícita do usuário.

## Fluxo

1. Confirmar **RT-XXX** (implementação segue o técnico).
2. Reler **CAT-*** e **RNG-*** do RN vinculado.
3. Seguir ordem do plano; escopo mínimo.
4. Atualizar seção **Implementação** no RT.
5. Sugerir `validar-entrega` (CAT + CANG).
