## Inclusão do Cliente — 6 abas (2026-06-01)

**Caminho:** Cadastros → Clientes → **Incluir** → Informação do Cliente.

### Aba Complemento

- **Cobrança:** endereço, cidade, estado, CEP.
- **Entrega:** endereço, cidade, estado, CEP (par separado).
- **Classe 1**, **Classe 2**, **Atividade** (lookup).
- **Cod.Ibge**, **Suframa**.
- Seção *Datas de Validade e Numeros de Alvara:* Validade Anvisa + Nº; Validade Mapa + Nº.

### Aba Responsaveis

Três blocos **Proprietario** (responsável legal, não veterinário):

- Proprietario, Observação, Nascimento, CPF, RG.

### Aba Referencias

- **Comerciais:** Empresa 1–3 + Telefone.
- **Bancárias:** Banco 1–2 + AG + CC + Telefone.

### Aba Financeiro *(defaults inclusão)*

| Campo | Valor |
|-------|-------|
| Cadastro | 01/06/2026 |
| LimiteC / Desconto / Comissão / Meta | 0,00 |
| Sistema | Prospecto |
| Juros | Não Cobrar Juros |
| Subs. Trib. | Calculo Externo |
| Motivo | Devolução Parcial |
| Venix / MixPedido | Enviar |
| Bloquear Portadores | Depósito, Boleto, Cheque |
| Venix e MixPedido | Dinheiro, Cartão |

Também: Serasa/Spc, Bloquear Fabricantes (lookup), Qual Motivo, Lembrete.

*Overlap:* Venix/MixPedido/portadores repetem na tela **MixPedido** (Alterar Informações) com Nova Senha e Crédito.

### Aba Diversos *(pet/vet)*

Dois blocos **Veterinario:** nome, Nascimento, CPF, RG, Crmv, Telefone.  
Campo **Crmv Emp** (empresa).
